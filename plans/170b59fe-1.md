## Current state

PR #4294 (`HexBerlekamp: prove Berlekamp matrix action identity`)
merged the feature slice for issue #4260. It proves the matrix-action
identity

```
(berlekampMatrix f hmonic) · coeffVector f w
  = coeffVector f (FpPoly.linearPow w p % f)
```

for monic `f`, prime `p`, and `w.size ≤ basisSize f` — the central
algebraic bridge between the Berlekamp matrix and the Frobenius
action on `F_p[X] / (f)`. Reading the progress file
`progress/20260515T193941Z_812eb95f.md`, the proof routes through a
private `matrixActionPolySum` packaging the column-scaled sum
`Σ_{j < n} C(w.coeff j) · powModMonic frobX f hmonic j` and combines
`FpPoly.compose_w_linearPow_X` with the existing column-entry
bridges in `HexBerlekamp/Basic.lean`.

The PR also promoted three Compose/Frobenius surfaces to public:

- `Hex.FpPoly.composeCoeffPowerSumUpTo` (definition).
- `Hex.FpPoly.compose_eq_coeff_power_sum_upTo_bound` (theorem).
- `Hex.FpPoly.powModMonic_pos_self_mod` (theorem).

This is the key prerequisite for `IsFixedSpaceKernelPolynomial f hmonic
w ↔ f ∣ FpPoly.linearPow w p - w` (#4261, currently claimed) and the
no-kernel-split irreducibility composition (#4227). The merged diff has
not yet had a dedicated review issue.
