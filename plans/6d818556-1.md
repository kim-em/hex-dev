`HexModArithMathlib` is a single 226-line file
(`HexModArithMathlib/Basic.lean`) in namespace
`HexModArithMathlib.ZMod64`. It is the Mathlib correspondence layer for
the executable `Hex.ZMod64 p` residue type: it transfers `ZMod64` to
Mathlib's `ZMod p` and proves the operations are homomorphic, packaged
as a ring equivalence.

The user-facing API surface is:

- `toZMod : Hex.ZMod64 p → ZMod p` and `ofZMod : ZMod p → Hex.ZMod64 p`
  — the two transfer maps.
- Round-trip lemmas `ofZMod_toZMod`, `toZMod_ofZMod` (the two inverse
  laws), and `val_toZMod`.
- Homomorphism-transfer lemmas: `toZMod_zero`/`ofZMod_zero`,
  `toZMod_one`/`ofZMod_one`, `toZMod_add`, `toZMod_neg`, `toZMod_sub`,
  `toZMod_mul`, `toZMod_natCast`, `toZMod_intCast`, `toZMod_pow`.
- `equiv : Hex.ZMod64 p ≃+* ZMod p` — the bundled ring equivalence,
  with `equiv_apply` / `equiv_symm_apply` unfolding it to
  `toZMod` / `ofZMod`.
