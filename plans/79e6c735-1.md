## Current state

`HexGF2/Clmul.lean` carries the pure-Lean carry-less multiplier
(`pureClmul`) and its one-hot/monomial specialisation lemmas. The
**public** API (`pureClmul`, `pureClmul_xor_left`, `pureClmul_oneHot*`,
`clmul`, `clmul_eq_pureClmul`, …) already has Phase 6 docstrings.

The **private helper cluster** that proves the in-word monomial
(one-hot) carry-less-multiply lemmas is still undocumented. These are
the lemmas in lines ~37–457 that build up to `pureClmul_oneHot`,
`pureClmul_oneHot_left_snd`, and `pureClmul_oneHot_left_fst`.
