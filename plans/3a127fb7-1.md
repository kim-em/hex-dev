Proof outline (for orientation only — choose any approach that closes the
goal): given `c : Vector R n` with `rowCombination M c = v`, the echelon
identity `M = D.transform * D.echelon` (or its existing equivalent in this
namespace) lets you transport `c` to coefficients `c' = transformInv * c`
against `D.echelon`. Pivot-row entries of `c'` are then forced to equal
`v[D.pivotCols.get pi] / D.echelon[pivotRow pi][D.pivotCols.get pi]` — which
is exactly the `echelonCoeffs` constructed by `spanCoeffs`. Reapplying the
transform yields `coeffs = transpose D.transform * echelonCoeffs`, the
`spanCoeffs` candidate; the `if rowCombination M coeffs = v then some coeffs
else none` guard then succeeds, so the result is `.isSome`.
