## Key decisions

- Used hypothesis-free functions (`steeredReduce`, the Bool checkers) in
  the worked example rather than `lll`/`firstShortVectorUnchecked`, which
  need `Rat` side-condition proofs — keeps the example robust and free of
  tactic-availability assumptions.
- Referenced the `GramSchmidt.Int` *namespace* in prose as a code span,
  not `{name}` (a namespace is not a constant; `{name}` errors on it).
- Cross-referenced the `HexLLLMathlib` soundness theorems
  (`certCheck_sound` etc.) in prose only — not `{name}` — to avoid pulling
  the Mathlib bridge into the Mathlib-free chapter's import surface.
