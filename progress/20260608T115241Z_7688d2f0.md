# 2026-06-08 11:52Z — sizeReduce size-reducedness (#6587)

## Accomplished

- Added `sizeReduce_size_reduced` to `HexLLLMathlib/Independent.lean`,
  closing sub-issue A of #6576.  After `LLLState.sizeReduce s k` for a
  valid state with independent basis, every Gram-Schmidt coefficient
  `μ[k][j]` (for `j < k`) satisfies `4 * μ * μ ≤ 1`.
- Supporting lemmas added in `HexLLLMathlib/Independent.lean`:
  - `nearestQuotient_residue_bound`: pure integer rounding bound
    `2 * |νjk - q * dj1| ≤ dj1` where `q = nearestQuotient νjk dj1`.
    Proved via `Int.fmod_def`, `Int.fmod_nonneg_of_pos`,
    `Int.fmod_lt_of_pos`, and `omega` on the residual identity.
  - `sizeReduceColumn_b_gramDet` / `sizeReduceColumn_independent`:
    a single `sizeReduceColumn` step preserves leading Gram dets
    (so `independent`) via `gramDet_sizeReduce`.
  - `sizeReduceColumn_ν_pivot_bound`: per-column pivot integer bound
    `2 * |ν[k][j]| ≤ d[j+1]` after `sizeReduceColumn` at pivot `j`.
    Reduce branch invokes `nearestQuotient_residue_bound`; no-op
    branch reads the failed integer test.
  - `sizeReduceColumn_ν_get_above_pivot`: preservation of `ν[k][l]`
    for `l.val > j.val` (the column-above-pivot case, off both the
    inner foldl's `<j` range and the `.set j` target).
  - `sizeReduce_foldl_preserves_ν_above` / `sizeReduce_foldl_d_get`:
    foldl-level preservation lemmas threaded through the standard
    `(List.finRange k).reverse` iteration.
  - `pairwise_finRange_reverse_lt`: the iteration list is `Pairwise`
    strictly decreasing in `Fin k.val`.
  - `sizeReduce_foldl_size_reduced`: foldl invariant combining
    per-column bound + preservation through the strictly decreasing
    tail (the iteration's structural property).
  - `sizeReduce_ν_bound`: integer formulation
    `2 * |ν'[k][j]| ≤ d[j+1]` for the full `sizeReduce` outer foldl.
  - `sizeReduce_size_reduced`: the rational formulation
    `4 * μ * μ ≤ 1`, via `scaledCoeffs_eq` bridging
    `ν' = d[j+1] * μ` (after positivity from independence).
