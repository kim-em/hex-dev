The public capstone is

```lean
theorem desnanot_jacobi_borderedMinor [CommRing R]
    (source : Hex.Matrix R n n) (k : Nat) (hk : k < n) (hnext : k + 1 < n)
    (i j : Fin n) (hi : k < i.val) (hj : k < j.val) :
    Hex.Matrix.det (Hex.Matrix.borderedMinor source (k + 1) hnext i j) *
        Hex.Matrix.det (Hex.Matrix.leadingPrefix source k _) =
      Hex.Matrix.det (Hex.Matrix.borderedMinor source k hk ⟨k,_⟩ ⟨k,_⟩) *
        Hex.Matrix.det (Hex.Matrix.borderedMinor source k hk i j) -
        Hex.Matrix.det (Hex.Matrix.borderedMinor source k hk i ⟨k,_⟩) *
        Hex.Matrix.det (Hex.Matrix.borderedMinor source k hk ⟨k,_⟩ j)
```

This is exactly the `hdesnanot` premise expected by
`bareissExactDiv_borderedMinor_of_mul_eq` with `prevPivot` instantiated
to `det (leadingPrefix source k _)`. Sign tracking on the `M_1k`/`M_k1`
permutations cancels by `Int.units_mul_self` (`(sign σ)² = 1`).
