## Current frontier

The two partial-prefix and partial-suffix expansion lemmas
(`det_columnSumMatrixWithPrefix_expand` and
`det_columnSumMatrixWithSuffix_expand`) are in place. What remains
is the actual all-column theorem
`det_columnSumMatrix_eq_sum_columnTuples` (deliverable 1 of #2878).
The skeleton is:

```lean
private theorem det_columnSumMatrixWithSuffix_eq_sum
    (source coeff : Matrix R n m) (chosen : List (Fin m))
    (hk : chosen.length ≤ n) :
    det (columnSumMatrixWithSuffix source coeff chosen) =
      (columnTupleVectors (n - chosen.length) m).foldl
        (fun acc pref => acc +
          partialColumnTupleCoeff coeff pref *
          det (columnTupleMatrix source (assembleColumnsSuffix chosen pref)))
        0
```

with `partialColumnTupleCoeff` and `assembleColumnsSuffix` defined as
private helpers. The induction is on `n - chosen.length`; the step
case uses `det_columnSumMatrixWithSuffix_expand` (LHS), the IH on
`c :: chosen` (inner sum), `foldl_det_sum_swap` (Fubini), the cTV
recursion on `(cTV (n - k) m)` (RHS), and identity matching of the
per-`(pref, c)` terms via the (yet-to-prove) lemmas
`partialColumnTupleCoeff_insertAt_last` and an analogous
`assembleColumnsSuffix_insertAt_eq`.
