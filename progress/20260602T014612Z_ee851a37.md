# HexMatrix: canonical raw nDet four-row Plucker identity (issue #6105)

## Accomplished

- Added private theorem `nDet_plucker_four_row_canonical` in
  `HexMatrix/Determinant.lean`. For strictly increasing
  `r0 < r1 < r2 < r3 : Fin (n + 2)`, the canonical three-term
  Plucker relation `nDet23 * nDet01 - nDet13 * nDet02 + nDet12 * nDet03 = 0`
  holds for the raw `nDet` minors of `B : Matrix R (n + 2) n`.
- Proof routes through the square two-row Plucker kernel
  `det_setRow_setRow_mul_det` applied to `M = nMatrix B r0 r1 h01` at the
  two replacement positions `s2 = ⟨r2.val - 2, _⟩`,
  `s3 = ⟨r3.val - 2, _⟩`. The four single-row transports
  (`det_setRow_nMatrix_{r2,r3}_{r0,r1}_eq_pow_mul_nDet_*`) and the
  double-row transport
  `det_setRow_setRow_nMatrix_r2_r0_r3_r1_eq_pow_mul_nDet_r2_r3` rewrite
  each of the five determinants in the square kernel to a signed `nDet`
  minor. The five sign exponents collapse against the common factor
  `(-1)^((r2 - r0 - 2) + (r3 - r1 - 2))` via `pow_add`, `pow_succ`, the
  parity identity `(r2 - r1 - 1) + (r3 - r0 - 2) =
  ((r2 - r0 - 2) + (r3 - r1 - 2)) + 1`, and `neg_one_pow_mul_self`.
- Dropped the `hraw` four-row hypothesis from the existing private helper
  `det_plucker_three_term_basisVec_of_gt_p3_of_nDet`, which now discharges
  it internally by calling `nDet_plucker_four_row_canonical` at
  `(p1, p2, p3, q)`. Moved the helper after the canonical theorem since
  the latter lives near the square Plucker kernel at the end of the file.
