loop downstream.

Argument:

1. Apply `bareissGramRowInvariant_noPivotLoop_initial` to the loop result,
   giving the represented row `v := rowCombination b (hinv.coeff sFin)` with
   coefficient supported on indices `≤ s`.
2. Show `Matrix.dot v (b.row k) = 0` for every `k.val ≤ s`. The `k.val = s`
   case is `h_zero` directly; the `k.val < s` case uses a new
   `noPivotLoop_matrix_processed_col_eq_zero` helper that proves every
   column processed by a regular Bareiss step is cleared below the pivot
   (and stays cleared by later iterations).
3. Expand `Matrix.dot v v` against the new rowCombination second-argument
   linearity identity `dot_rowCombination_right_eq_foldl_int`. Every
   summand is zero: either the coefficient is zero (k.val > s, by
   `coeff_supp`) or the dot is zero (k.val ≤ s, by step 2).
4. Conclude `Matrix.dot v v = 0`. By integer positive definiteness
   (`int_dot_eq_zero_of_dot_self_zero_left`), any dot against `v` vanishes.
5. Transport across symmetry: `noPivotLoop_matrix_symm_preserve` plus a
   local invocation of `gramMatrix_getElem` + `int_dot_comm_local` gives
   `state.matrix[i][sFin] = state.matrix[sFin][i]`, which equals
   `Matrix.dot v (b.row i)` by the invariant, hence zero.
