### Implementation (#4706)

Added `HexHenselMathlib.isCoprime_cancel_of_natDegree_lt` to
`HexHenselMathlib/Basic.lean`. The statement is:

```
theorem isCoprime_cancel_of_natDegree_lt
    {R : Type*} [CommRing R] [IsDomain R]
    {g h a b : Polynomial R} (hg : g.Monic) (hcop : IsCoprime g h)
    (heq : a * h + b * g = 0)
    (hdeg : a.natDegree < g.natDegree) :
    a = 0 ∧ b = 0
```

Proof: `a * h = g * (-b)` (linear_combination from `heq`), so
`g ∣ a * h`; by `IsCoprime.dvd_of_dvd_mul_right`, `g ∣ a`; by
`Polynomial.eq_zero_of_dvd_of_natDegree_lt` (strict degree bound),
`a = 0`; substituting back, `b * g = 0`, and `g ≠ 0` from `Monic.ne_zero`
gives `b = 0`.

Three Mathlib imports added to `HexHenselMathlib/Basic.lean`:
`Mathlib.Algebra.Polynomial.Degree.Domain`,
`Mathlib.Algebra.Polynomial.Monic`,
`Mathlib.Tactic.LinearCombination`.
