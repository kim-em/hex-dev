### Helper theorem (Mathlib bridge)

New theorem appended to `HexBerlekampZassenhausMathlib/IntReductionMod.lean`
(just before the slow-path quadratic arm umbrella, both outside the
`IntReductionMod` namespace):

```
HexBerlekampZassenhausMathlib.quadraticIntegerRootFactors?_residual_irreducible_of_isPrimitive
    {core : Hex.ZPoly} {coreFactors : Array Hex.ZPoly}
    (hprim : (HexPolyZMathlib.toPolynomial core).IsPrimitive)
    (hcore_lc_pos : 0 < Hex.DensePoly.leadingCoeff core)
    (hquad : Hex.quadraticIntegerRootFactors? core = some coreFactors)
    (hres_ne_one :
      (Hex.splitIntegerRootFactorsAux core
        (Hex.integerRootCandidates core)
        (Hex.integerRootCandidates core).length).2 ≠ 1) :
    Hex.ZPoly.Irreducible
      (Hex.splitIntegerRootFactorsAux core
        (Hex.integerRootCandidates core)
        (Hex.integerRootCandidates core).length).2
```

Proof structure (`sorry`-free):

* Use the freshly-public `Hex.splitIntegerRootFactorsAux_product` to
  obtain `split.2 * polyProduct split.1 = core`, and
  `Hex.splitIntegerRootFactorsAux_polyProduct_leadingCoeff_pos` to
  obtain `0 < leadingCoeff (polyProduct split.1)`.  Combined with
  `hcore_lc_pos` and `ZPoly.leadingCoeff_mul_of_nonzero`, conclude
  `0 < leadingCoeff split.2`.
* Transport `split.2 * polyProduct split.1 = core` through
  `HexPolyZMathlib.toPolynomial_mul` to get
  `toPolynomial split.2 ∣ toPolynomial core`, then apply
  `Polynomial.isPrimitive_of_dvd hprim` to inherit primitivity.
* Unfold `Hex.quadraticIntegerRootFactors?` against `hquad` to
  extract `split.2.degree?.getD 0 ≤ 1`.  Combined with
  `HexPolyMathlib.natDegree_toPolynomial`, get
  `(toPolynomial split.2).natDegree ≤ 1`.
* Case split on `natDegree`:
  * Degree 0: `Polynomial.eq_C_of_natDegree_eq_zero` + the
    `isPrimitive_iff_isUnit_of_C_dvd` mp direction give the constant
    is a unit (so `±1` in `ℤ`); the positive leading coefficient
    (`hres_lc_pos`) rules out `-1`, leaving `split.2 = 1` which
    contradicts `hres_ne_one`.
  * Degree 1: `Polynomial.degree_map_eq_of_injective` (via
    `Int.cast_injective`) preserves degree under the cast `ℤ → ℚ`;
    `Polynomial.irreducible_of_degree_eq_one` gives irreducibility
    over `ℚ[X]`; Gauss's lemma
    `Polynomial.IsPrimitive.Int.irreducible_iff_irreducible_map_cast`
    lifts back to `ℤ[X]`; the existing
    `Hex.ZPoly.polynomialIrreducible_iff_irreducible` bridge then
    yields `Hex.ZPoly.Irreducible split.2`.

### Wiring into the slow-quadratic umbrella

The residual sub-case `sorry` at
`HexBerlekampZassenhausMathlib/IntReductionMod.lean:1258` (added by
PR #4580 just before this session started) is replaced with an
application of the helper.  All three preconditions discharged
inline:

* `hprim`: `IntReductionMod.normalizeForFactor_squareFreeCore_toPolynomial_isPrimitive f _hf_ne`.
* `hcore_lc_pos`: `Hex.ZPoly.leadingCoeff_squareFreeCore_nonneg`
  upgraded to strict positivity using `hdeg`
  (`squareFreeCore.degree?.getD 0 ≠ 0` ⟹ `squareFreeCore ≠ 0` ⟹
  `leadingCoeff ≠ 0` via `Hex.ZPoly.leadingCoeff_ne_zero_of_ne_zero`).
* `hres_ne_one`: by contradiction.  If the residual were `1`, then
  `raw = 1`, so `entry.1 = normalizeFactorSign 1 = 1`, contradicting
  `Hex.factorWithBound_entry_shouldRecord` (which forbids `entry.1 = 1`).

The umbrella's docstring is updated to reflect the discharged
status.
