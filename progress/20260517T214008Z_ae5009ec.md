`int_factor_one_eq_unit` helper, before `irreducible_of_size_two_monic`):

* `isNatPrime_iff` (private): the elementary characterisation
  `isNatPrime n = true ↔ 2 ≤ n ∧ ∀ d, 2 ≤ d → d * d ≤ n → ¬ d ∣ n`.
  Forward direction handles the bool/Prop transitions explicitly via
  `Bool.and_eq_true`, `decide_eq_true_iff`, `Bool.not_eq_true'`, and
  `List.any_eq_false` (the latter gives `¬ pred = true`, not `pred = false`),
  then discharges the `d < n` obligation arithmetically from
  `2 * d ≤ d * d ≤ n` plus omega.
* `zpoly_C_mul_C` (private support): `C (a * b) = C a * C b` over `Int`,
  proved by `ext_coeff` + `coeff_scale` + case split on `n = 0`.
  Existing `C_mul_eq_scale` did `C c * p = scale c p` but no `C * C = C *`
  lemma existed.
* `zpoly_size_C_of_ne_zero` (private support): `(C k).size = 1` for
  `k ≠ 0` via `coeffs_C_of_ne_zero` + `rfl`. The existing
  `size_C_le_one` only gives `≤ 1`.
* `irreducible_C_of_isNatPrime` (private): `isNatPrime k.natAbs = true →
  ZPoly.Irreducible (DensePoly.C k)`. The `no_factors` case uses
  `by_cases` on `ZPoly.IsUnit a/b` (decidable via `instDecidableIsUnit`)
  instead of `by_contra/push_neg` (which are Mathlib tactics not available
  in this Mathlib-free file). Closes via the WLOG min(|a₀|, |b₀|) divisor
  of `|k|` argument against `isNatPrime`'s elementary divisor exclusion.
* `isNatPrime_natAbs_of_irreducible_C` (private): `k ≠ 0` and
  `ZPoly.Irreducible (DensePoly.C k) → isNatPrime k.natAbs = true`.
  Splits on sign of `k` (via `Int.natAbs_eq`), constructs `C k = C (±d) * C e`
  from the natural divisor split `|k| = d * e`, and reads off the
  contradiction with `no_factors` since both `(±d : Int)`, `e ≥ 2`.
