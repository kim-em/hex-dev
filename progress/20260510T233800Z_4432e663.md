# 4432e663 — feature: primitive polynomial scalar annihilator (#3168)

## Accomplished

Added `nat_dvd_of_scalar_mul_primitive_coeff_dvd` to `HexPoly/Euclid.lean`:
if `content p = 1` and `(d : Int) ∣ a * p.coeff n` for every `n`, then
`(d : Int) ∣ a`. The proof combines the Bezout certificate from #3173
(`exists_linear_combination_coeffs_eq_one_of_content_eq_one`) with two
new private helpers:

- `foldl_zipWith_mul_scale_int` — distributes scalar multiplication
  through `(zipWith (· * ·) ...).foldl (+) 0`.
- `dvd_foldl_zipWith_scale_mul` — divisibility through the same
  scaled foldl sum when each value satisfies `d ∣ a * c`.
