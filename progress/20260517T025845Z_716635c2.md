### New theorems in `IntReductionMod.lean`

Four auxiliary bridges followed by one main theorem, placed
immediately after `normalizeForFactor_squareFreeCore_toPolynomial_isPrimitive`
(per the issue's "adjacent to" / "auxiliary theorems immediately above
the main theorem" layout instructions):

1. `toPolynomial_derivative {R} [CommSemiring R] [DecidableEq R]` —
   generic semiring-level identity: `HexPolyMathlib.toPolynomial`
   intertwines the executable `Hex.DensePoly.derivative` with
   `Polynomial.derivative`.

2. `toPolynomial_toRatPoly_eq_map_intCast (f)` — the rational view of
   an integer polynomial through `Hex.ZPoly.toRatPoly` agrees with the
   integer-side `HexPolyZMathlib.toPolynomial` post-composed with
   `Polynomial.map (Int.castRingHom ℚ)`.

3. `isCoprime_toPolynomial_map_intCast_derivative_of_squareFreeRat
   (f) (hf : f ≠ 0) (hsf : SquareFreeRat f)` — the executable
   square-free invariant transports to `IsCoprime` of the rational
   image with its formal derivative, via
   `HexPolyMathlib.toPolynomial_gcd_associated` over the field `ℚ`.
   The executable `(gcd …).size ≤ 1` plus `f ≠ 0` forces the
   executable gcd to be a nonzero constant `C c`, hence a unit in
   `Polynomial ℚ`; associatedness lifts unit-ness to the Mathlib gcd,
   which is then `IsCoprime` via `EuclideanDomain.gcd_isUnit_iff`.

4. `squarefree_of_isPrimitive_of_squarefree_map_intCast` — Gauss-style
   descent: a primitive `p : Polynomial ℤ` is squarefree whenever
   `p.map (Int.castRingHom ℚ)` is.  A square factor `q² ∣ p` lifts to
   `(q.map …)² ∣ p.map …`, squarefreeness pins `q.map …` to a unit
   (hence `natDegree 0`), the injective cast (`Int.cast_injective`)
   pulls back to `q.natDegree = 0`, so `q = C n` with
   `C (n * n) ∣ p`; primitivity then forces `IsUnit (n * n)`, hence
   `IsUnit n` (via `isUnit_pow_iff`), hence `IsUnit (C n) = IsUnit q`.

5. **Main:**
   `normalizeForFactor_squareFreeCore_toPolynomial_squarefree
   (f : Hex.ZPoly) (hf : f ≠ 0)` —
   `Squarefree (HexPolyZMathlib.toPolynomial (Hex.normalizeForFactor f).squareFreeCore)`.

   Proof composes: `(normalizeForFactor f).squareFreeCore ≠ 0` from
   primitivity of the product (existing
   `primitiveSquareFreeDecomposition_squareFreeCore_repeatedPart_primitive`),
   then `SquareFreeRat` of the core via
   `primitiveSquareFreeDecomposition_squareFreeCore`, then
   `IsCoprime` via (3), then `Squarefree` of the rational image via
   `Polynomial.Separable.squarefree` (recall `Separable := IsCoprime
   _ _.derivative`), then descent to `Polynomial ℤ` via (4) using
   the existing primitivity bridge.
