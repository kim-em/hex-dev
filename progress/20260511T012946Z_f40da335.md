# HexMatrix Determinant: canonical sort + orbit factorization (#3179)

## Accomplished

Implemented and proved the canonical sort/permutation factorization for
injective column tuples in `HexMatrix/Determinant.lean`, in support of
the Cauchy-Binet orbit-grouping lemma `det_gramMatrix_eq_sum_minors_sq`
(#3180). The added API is:

- `sortInjPerm cols : Vector (Fin n) n` (rank-based permutation, total).
- `sortInjTuple cols : Vector (Fin m) n` (canonical sorted version, total).
- `reconstructInjTuple sel perm : Vector (Fin m) n` (inverse map).
- Membership theorems: `sortInjPerm_mem_permutationVectors`,
  `sortInjTuple_mem_selectedColumnTuples`,
  `isStrictlyIncreasingColumnTuple_sortInjTuple`.
- Factorization equation `cols_getElem_eq_sortInjTuple_sortInjPerm`.
- Forward injectivity (pairwise distinctness): `sortInj_pair_injective`.
- Reconstruction-side: `reconstructInjTuple_injective`,
  `sortInjPerm_reconstructInjTuple`, `sortInjTuple_reconstructInjTuple`,
  `reconstructInjTuple_sortInj`.
- List-form bijection: `selPerm_reconstructed_list_nodup` and
  `mem_selPerm_reconstructed_iff`. These pair to give the
  flat-list-Perm characterization that #3180 can consume directly.

Implementation chose the rank-based encoding (count of strictly-smaller
positions) for `sortInjPerm`; this keeps the proofs structural and
avoids introducing an insertion-sort routine. `sortInjTuple` is defined
through the existing `inversePermutationVector` so it is total without
classical choice.
