# Split HexMatrix → HexMatrix / HexRowReduce / HexDeterminant / HexBareiss (structural)

## Accomplished

First PR of the HexMatrix split: the **structural computational split**. The
~18k-line `HexMatrix` library is divided into four Mathlib-free libraries along
the dependency seams, with the target DAG (`hex-determinant` does **not** depend
on `hex-row-reduce`):

```
HexMatrix (base) ── HexRowReduce
                 └── HexDeterminant ── HexBareiss
```

- **Phase 0**: severed the determinant→row-reduce edge. `HexDeterminant/Leibniz`
  imported `HexMatrix.RowEchelon` but used none of its declarations — only its
  transitive `Batteries.Data.List.Perm`/`Pairwise` and the base algebra modules.
  Replaced with direct `Batteries.Data.List.Perm`/`Pairwise` +
  `HexMatrix.Elementary`/`DotProduct`/`MatrixAlgebra` imports.
- **Phase 1**: `git mv` of RowEchelon/RREF → `HexRowReduce/`, Determinant/* →
  `HexDeterminant/` (inner nesting collapsed), Bareiss + BorderedMinor →
  `HexBareiss/`. Repo-wide import rewrite (`HexMatrix.RowEchelon`→
  `HexRowReduce.RowEchelon`, etc.); internal `namespace Hex.Matrix.*` left
  unchanged so symbol references downstream are untouched. New umbrellas
  `HexRowReduce.lean`/`HexBareiss.lean`; trimmed `HexMatrix.lean` to base.
  Consumers importing the old umbrella switched to the specific split libs
  (`HexBerlekamp/Basic` += `HexRowReduce`; `bench/HexMatrix/Bench` +=
  `HexDeterminant`,`HexBareiss`).
- **Phase 2**: three `lean_lib` targets in `lakefile.lean`; `libraries.yml`
  entries (deps `HexRowReduce→[HexMatrix]`, `HexDeterminant→[HexMatrix]`,
  `HexBareiss→[HexDeterminant,HexMatrix]`) and updated consumer deps
  (`HexGramSchmidt`, `HexBerlekamp`, `HexLLL`, `HexMatrixMathlib`). Interim
  per-lib Phase-4 reports added.

Verified green: full `lake build` (HexManual + all libs, 2671 jobs), the four
computational libs, all downstream consumers (incl. Mathlib bridges),
bench/conformance exes; conformance fixtures byte-identical; validators
`check_dag`, `check_phase4` (CI mode), `check_copyright_headers`,
`conformance_targets --check`, `check_benches_mathlib_free` all pass.
