# #6744 steered reducer — feasibility validated (prototype)

## Accomplished

Built a standalone prototype (`HexLLL/SteeredProto.lean`, scratch, not
shipped) of the approximation-steered reducer: exact integer row ops on the
basis, untrusted Float Gram-Schmidt steering (Schnorr-Euchner / Cohen 2.6.3
float swap formulas), per-row drift-free refresh of μ/B from the exact Gram
entries `⟨b_i,b_j⟩` (recomputed from `b`, no stored Gram), a drift-free final
size-reduction sweep, and exact (δ,11/20) certification.

Baseline captured on carica (merge-base 6f347b5): harsh n=55 = 242ms,
random n=180 = 4.76s region. Local machine ≈ carica (harsh n=55 native
240ms, random n=180 native 4.62s).

## Key results (local, ≈carica), steer δ=9/10, certify δ=3/4, refresh period 16

- harsh: n=25 1.7ms, n=40 ~4ms, n=55 13.4ms vs native 240ms → ~18×,
  fitted exponent ~2.6 (≪ 4.5). 0 swaps (diagonal-dominant); win is skipping
  the Θ(n⁴)-bit exact determinant state. First-row NormSq UNCHANGED.
- random: n=30 7.1ms (native 15.5 → 2.2×, NO low-rung regression),
  n=60 48ms (3.0×), n=90 149ms (3.4×), n=180 1217ms (4624 → 3.8×).
- ALL rungs certify (δ,11/20) exactly → fallback 0.

## Critical engineering findings (must carry into production)

1. In-place mutation is essential. Nested `Array (Array Float)` μ with naive
   `set!` COWs whole rows under swap (O(n²)/swap) → random n=180 ~2.3s
   (barely 2×). Using `Array.modify` (placeholder-swap, the same pattern as
   the existing `LLLState.swapStep`) gives genuine in-place updates →
   1217ms (3.8×). Production must use `Vector.modify`/`Array.modify`.
2. Steer at a δ STRICTER than the certify δ (e.g. (δ+1)/2). Steering at the
   certify δ=3/4 left random n=30 with a Lovász ratio that float accepted but
   exact 3/4 rejected → certification fail. δ_steer=9/10 fixes it (margin
   absorbs float slop). Cost: more swaps, still comfortably ≥2×.
3. Drift-free refresh of the working row from exact Gram each visit (or every
   K visits) + a final drift-free size-reduction sweep guarantee the output
   is genuinely (δ,11/20)-reduced regardless of incremental float drift.
4. Float never overflows through the ladder (norms < 2^1024).
