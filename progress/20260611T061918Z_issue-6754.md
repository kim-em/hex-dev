## Branch-1 soundness (directive caveat)

The directive flagged branch-1 (degree-0 core) as a possible soundness gap:
`reassemblePolynomialFactors d #[d.squareFreeCore]` with `squareFreeCore = 1`
appends the unit `1` to the raw array, and `Irreducible 1` is false — so the
raw-level `h_raw` is literally false for branch-1.

Resolution: branch-1 is **unreachable** under the `h_raw` dispatch
hypotheses. `factorFastFactorsWithBound` (and `factorSlowModularWithBound`)
short-circuit with `some _` whenever the square-free core is degree-0, so the
slow-trial disjunct's `factorFastFactorsWithBound f B = none` premise forces
the core to have positive degree. The helper
`factorFast_none_squareFreeCore_degree_ne_zero` encodes this; the main theorem
takes only `fast = none` (the strictly weaker premise sufficient to exclude
branch-1). No statement weakening, no sorry/axiom.
