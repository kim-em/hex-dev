## Approach

Strong induction on `n - d`, structured as a recursive function
pattern-matching on the bound. At each step:
- If there exists a strictly larger proper divisor `e` of `n` with
  `d ∣ e`, recurse on `e` (decreasing `n - e`).
- Otherwise `d` itself is maximal: there is no `e ∈ properDivisors n`
  with `d < e ∧ d ∣ e`, which is exactly the negation of what
  `maximalProperDivisors`'s filter excludes.

The boolean reasoning to discharge the maximality leg avoids `by_contra`
(unavailable in this file's import surface) by using
`Bool.eq_false_iff.mpr` and `List.any_eq_true`.
