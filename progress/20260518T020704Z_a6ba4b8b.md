## Accomplished

- Added `Hex.ZPoly.Irreducible_of_modP_irreducible_of_primitive_of_admissible`
  to `HexBerlekampZassenhaus/Basic.lean`: given a primitive `ZPoly` `f` with
  `1 < f.size`, a prime `p`, `leadingCoeffAdmissible f p`, and
  `FpPoly.Irreducible (modP p f)`, conclude `ZPoly.Irreducible f`.
- Added a private helper `Hex.size_modP_le` (a sibling of the existing
  `*_modP_eq_of_leadingCoeffAdmissible` substrate) stating
  `(ZPoly.modP p f).size ≤ f.size`. The existing
  `size_modP_eq_of_leadingCoeffAdmissible` proof inlines the same argument; the
  new helper exposes the upper-bound half for reuse without disturbing the
  existing wrapper.
- Proof structure mirrors
  `HexBerlekampZassenhausMathlib.irreducible_of_isPrimitive_of_irreducible_map_intCast_zmod`
  (`HexBerlekampZassenhausMathlib/IntReductionMod.lean:106`). The
  Mathlib-free version uses size-arithmetic over `FpPoly.size_mul_eq_add_sub_one`
  (no-zero-divisors over a prime modulus) plus the new substrate's
  `size_modP_eq_of_leadingCoeffAdmissible` to force `(modP p a).size = a.size`
  and `(modP p b).size = b.size`, then transfers `degree? = some 0` back to
  `a.size = 1` (or `b.size = 1`). Primitivity (`content f = 1`) then closes the
  constant case via `dvd_content_of_nat_dvd_coeff`, forcing `|coeff 0| = 1` and
  hence `ZPoly.IsUnit`.
- The `hnotConstant : 1 < f.size` precondition is required: the executable
  `FpPoly.Irreducible` predicate only asserts the factorization disjunction,
  whereas Mathlib's `_root_.Irreducible` excludes constants implicitly via
  `not_isUnit`. This is documented in the doc-string.
