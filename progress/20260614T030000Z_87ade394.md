## Key finding (recorded on the PR)

The proof is **vacuously true**: it derives `f' = 0` from
`c.size = 1 ∧ f' ≠ 0` and closes with `exfalso`. That is the correct
math — when `f' ≠ 0`, `c = f / gcd(f, f')` has `deg c ≥ 1`
(`deg g ≤ deg f' < deg f`), so a size-one quotient is impossible, not
merely non-one. This rules out the "scalar trap" more strongly than the
issue framed it. Implication for #6904: the size-one base case never
fires in the derivative-active branch; the recursive step should not
expect a constructive size-one witness, and the obligation is
dischargeable from this lemma's signature alone. `c.size ≠ 1` would be a
clearer statement, but the requested `c.size = 1 → c = 1` form satisfies
the contract and composes.
