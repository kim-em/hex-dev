## Accomplished

Landed the concrete bridge-side instance of `Hex.GramSchmidt.Int.StepWitness b`
for arbitrary integer bases in `HexGramSchmidtMathlib/Int.lean`:

- **`StepWitness.ofGram b : Hex.GramSchmidt.Int.StepWitness b`** — public,
  sorry-free. Quotient at slot `a` is the next iteration's canonical
  coefficient `(bareissGramCanonicalCoeff b (fuel + 1) i)[a]`. Integrality is
  routed through the augmented Bareiss-Desnanot identity:
  - lift Gram-side state matrix entries and canonical coefficients to the
    `augmentedGram b a` no-pivot Bareiss trajectory via
    `noPivotLoop_augmentedGram_invariant`;
  - identify each augmented-side entry with a bordered-minor determinant via
    `trailing_eq_at_step_local` and the augmented `BareissNoPivotInvariant`;
  - apply `HexMatrixMathlib.desnanot_jacobi_borderedMinor` at level `fuel` to
    obtain the multiplicative form
    `det B(fuel+1) * det (leadingPrefix fuel) = det B(fuel, K, K) * det B(fuel, i, L) - det B(fuel, i, K) * det B(fuel, K, L)`;
  - identify the augmented-side bordered minor at `fuel + 1` with the next
    canonical coefficient via `bareissGramCanonicalCoeff_eq_borderedMinor_aug`.
- **`noPivotLoop_initial_step_eq_and_fuel_succ_le_of_no_sing`** (private,
  helper) — under a non-singular Bareiss prefix that has not reached the
  terminal step, the loop consumed exactly `fuel` regular iterations
  (`state.step = fuel ∧ fuel + 1 ≤ n`). Inducts on fuel with case analysis
  on the singular / regular / done branch.
- **`prevPivot_eq_at_step_local`** (private, helper) — companion to
  `trailing_eq_at_step_local` for the `prevPivot_eq` field of
  `BareissNoPivotInvariant`, supplied with the step value externally.
