# Failed Bareiss column adjugate dependence — issue #2773

## Accomplished

Closed issue #2773 by adding the failed-pivot column dependence helper to
`HexMatrixMathlib/Determinant/Bareiss.lean`.

The new public surface:

- `failed_bareiss_column_dependence` — main theorem: when the row-pivoted
  Bareiss singular branch reports a zero `k`-bordered trailing pivot column
  and the size-`k` leading-prefix determinant is nonzero, there is an
  explicit coefficient function `α : Fin n → Int` with `α ⟨k, hk⟩ ≠ 0`,
  zero on columns past `k`, and `(matrixEquiv source).mulVec α = 0`.
- `failedBareissColumn` — the noncomputable witness coefficient function,
  defined via `(-1)^(k + c.val)` times the signed cofactor of the
  leading-prefix-augmented top block with column `c` removed.
- `failedBareissColumn_at_pivot` — the coefficient on column `k` is exactly
  `Hex.Matrix.det (Hex.Matrix.leadingPrefix source k _)`.
- `failedBareissColumn_above_pivot` — coefficient is zero past column `k`.

Supporting private lemmas: `failedBareissTopBlock`, top-block apply lemmas,
the `(Fin.last k).succAbove` reduction, the bordered-minor `(last, c')`
entry formula, the cofactor-minor / top-block submatrix equality, the
cofactor-expansion identity `failedBareissColumn_dot_row_eq_borderedMinor_det`,
and the duplicate-row determinant zero lemma for `r.val < k`.

The proof uses the standard cofactor/Cramer trick: cofactor expansion along
the last row of `matrixEquiv (borderedMinor source k _ r ⟨k,_⟩)` via
`Matrix.det_succ_row` at `Fin.last k` produces coefficients independent of
`r`. Splitting the source-row sum into `r.val < k` (duplicate row in the
bordered minor → `Matrix.det_zero_of_row_eq`) and `r.val ≥ k` (failed-column
hypothesis → `det = 0`) gives `(matrixEquiv source).mulVec α = 0`.
