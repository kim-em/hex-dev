Landed the redesign half of Stage P in `HexGramSchmidt/Int.lean`:

- New pure recursive function
  `Hex.GramSchmidt.Int.bareissGramCanonicalCoeff (b : Matrix Int n m) :
  Nat → Fin n → Vector Int n` capturing the canonical row-coefficient vector
  of the initial no-pivot Gram trajectory, with recursion equations for the
  four loop branches (regular active row, regular processed row, singular,
  done) all proved by definitional unfolding.
- Canonicity predicate
  `IsCanonicalAt (b) (fuel) (hinv : BareissGramRowInvariant b
  (noPivotLoop fuel (noPivotInitialState (gramMatrix b)))) : Prop` defined
  as `∀ i, hinv.coeff i = bareissGramCanonicalCoeff b fuel i`. Base-case
  initial-invariant canonicity proved as `isCanonicalAt_initial`.

  `bareissGramRowInvariant_regular_step_coeff_canonical` showing the
  invariant produced by `bareissGramRowInvariant_regular_step` from a
  canonical input matches `bareissGramCanonicalCoeff` at the next-step fuel

  next-step witness.
- Transport-invariance helper `bareissGramRowInvariant_coeff_transport`
  showing `(h ▸ x).coeff i = x.coeff i` for `h : st1 = st2` — needed

## Next step

A successor session picks up the bridge-side instance construction. The
acceptance criterion: a sorry-free public theorem in `HexGramSchmidtMathlib`
of shape

```
theorem stepWitness (b : Matrix Int n m) : Hex.GramSchmidt.Int.StepWitness b
```

The proof route is Bareiss-Desnanot on the PSD Gram minors: the witness
`q a := bareissGramCanonicalCoeff b (fuel + 1) i a` (the next iteration's
canonical coefficient), and divisibility follows from identifying
`bareissGramCanonicalCoeff` at iteration `e` for active row `i`, slot `a`,
with a specific bordered/Cramer minor determinant of `b`, then applying the
standard Bareiss exact-division identity on the augmented Bareiss trajectory.
Existing infrastructure: `noPivotLoop_full_eq_borderedMinor_det`
([HexMatrixMathlib/Determinant/Bareiss.lean](../HexMatrixMathlib/Determinant/Bareiss.lean)),
`scaledCoeffMatrix_det_eq_gramDet_mul_coeffs`
([HexGramSchmidtMathlib/Int.lean:1378](../HexGramSchmidtMathlib/Int.lean#L1378)),
`scaledCoeffs_eq_scaledCoeffMatrix_bareiss` (sorry-free per #6517 + #6519).
