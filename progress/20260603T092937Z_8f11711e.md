inequality, decomposing the canonical product LHS
`coeffL2NormBound^aux.natDegree * auxiliaryBound^input.natDegree` against
`bhksPaperThresholdReal core C` into two independent sub-bounds.

  `HexBerlekampZassenhausMathlib/BHKSBound.lean` — pure paper-threshold

  `auxPow ≤ bhksPaperDegreeFactorReal f * bhksPaperConstantFactorReal f C *
  bhksPaperLogFactorReal f` (plus `0 ≤ auxPow`), concludes
  `coeffPow * auxPow ≤ bhksPaperThresholdReal f C` via `mul_le_mul` plus a
  `ring` rearrangement.

  `HexBerlekampZassenhausMathlib/Recovery.lean` — chain-level wrapper. Takes
  the factored bounds and feeds them through
  `bhksPaperThresholdReal_ge_of_factored_bounds`, then composes with the
  existing `bhksPaperThresholdReal_chain_lt_p_pow_kLocalFactorDegree` to
  yield the strict `< p^(k * localFactorDegree)` cap-arithmetic comparison.

These are pure compositions; no new analytic content.

## Current frontier

A caller bounding the BHKS §5 Theorem 5.2 LHS now has two routes:

1. **Direct route** — prove the full
   `coeffL2NormBound^aux.natDeg * auxiliaryBound^input.natDeg ≤
   bhksPaperThresholdReal core C` inequality and feed it through
   `bhksPaperThresholdReal_chain_lt_p_pow_kLocalFactorDegree` (existing
   path used by the `*AndPaperThreshold` constructors).
2. **Factored route** (new) — prove `coeffL2NormBound^aux.natDeg ≤
   bhksPaperCoeffNormFactorReal core` and `auxiliaryBound^input.natDeg ≤
   bhksPaperDegreeFactorReal core * bhksPaperConstantFactorReal core C *
   bhksPaperLogFactorReal core` independently and feed both through
   `bhksPaperThresholdReal_chain_lt_p_pow_kLocalFactorDegree_factored`.

The factored sub-bounds match the natural BHKS shape: the
coefficient-power piece bounded by `‖core‖₂^(2n-1)`, and the
auxiliary-power piece bounded by `n · (2C)^(n²) · (log ‖core‖₂)^n`.
