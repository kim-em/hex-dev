# Partition matrix bench / conformance / SPEC / Phase-4 by operation family (PR2)

## Accomplished

Follow-up to the structural split (PR #8418): partitioned the shared
bench/conformance/SPEC/Phase-4 artefacts so each split library owns its own,
keeping the released repos self-contained.

- **Bench** split three ways: `bench/HexMatrix/Bench.lean` (dense
  multiplication only), `bench/HexDeterminant/Bench.lean` (`runLeibnizDet`),
  `bench/HexBareiss/Bench.lean` (`runBareissDet` + the FLINT comparator ladder).
  New `hexdeterminant_bench` / `hexbareiss_bench` exes; ci.yml builds all three
  and smoke-verifies the cheap `hexdeterminant_bench` (matrix/bareiss stay
  build-only — dense fixtures / heavy FLINT rungs).
- **Conformance** split four ways. The `#guard` module became base
  (`HexMatrix`, oracle-free) + `HexRowReduce` (rref/span/nullspace) +
  `HexDeterminant` (det) + `HexBareiss` (bareiss). Three self-contained emit
  drivers replace the single `hexmatrix_emit_fixtures`, each emitting the shared
  matrices with only its ops. The op-keyed `matrix_flint.py` oracle is reused
  unchanged across all three per-lib fixture streams; `run_oracles.sh` and
  `conformance.yml` updated. Oracle verified locally with python-flint 0.8.0
  (rowreduce 27, determinant 9, bareiss 9 cases, 0 failures; emit==committed).
- **Phase-4 metadata** reassigned in `libraries.yml`: `HexMatrix` keeps
  `dense-square-multiplication`; `HexDeterminant` gets `leibniz-small-determinant`;
  `HexBareiss` gets `structured-bareiss-determinant` + the `FLINT fmpz_mat_det`
  informational comparator; `HexRowReduce` has no perf bench (conformance-only).
- **Reports** rewritten per lib (real measurements, updated target names); the
  consolidated `hex-matrix-performance.md` trimmed to the multiplication surface.
- **SPEC** split into `HexMatrix/SPEC/hex-matrix.md` (base),
  `HexRowReduce/SPEC/hex-row-reduce.md`, `HexDeterminant/SPEC/hex-determinant.md`,
  `HexBareiss/SPEC/hex-bareiss.md`.

Verified green: `lake build` (2784 jobs), the three bench exes, the conformance
lib + three emit exes, `hexdeterminant_bench verify`; validators `check_dag`,
`check_phase4`, `check_copyright_headers`, `conformance_targets --check`,
`check_benches_mathlib_free` (18 bench exes, mathlib-free).
