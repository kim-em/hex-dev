final residual", is false for repeated factors because `trialDivisionPeelAux`
tries each candidate only once. For example, a target containing `c^2` can emit

divisor hypothesis is threaded in.

square-free core context to "an emitted candidate cannot also divide the final
residual" (or an equivalent no-square-divisor statement), plus the degree-half
