```lean
theorem Hex.ZPoly.coeffL2NormBound_sq_le_two_mul_coeffNormSq (f : Hex.ZPoly) :
    (Hex.ZPoly.coeffL2NormBound f) ^ 2 ≤ 2 * Hex.ZPoly.coeffNormSq f
```

The proof uses the new `floorSqrt_sq_le` theorem from #6715. In the perfect
square branch, `ceilSqrt = floorSqrt`; in the non-perfect branch, it uses
`floorSqrt n ^ 2 < n` and the elementary bound
`(floorSqrt n + 1)^2 ≤ 2n`.
