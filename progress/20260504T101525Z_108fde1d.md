- Decomposed parent #1973 into:
  - #2068 (this PR): the parametric `Matrix.identity_independent` helper
    plus its `gramMatrix_one` / `submatrix_one` supporting lemmas;

- Added `Matrix.getElem_one : (1 : Matrix R n n)[i][j] = if i = j then 1 else 0`
  in `HexMatrix/Basic.lean` as a simp lemma. Trivial proof unfolding
  `Matrix.identity` and `Matrix.ofFn`.
- Added supporting foldl helpers in `HexMatrix/Basic.lean`:
  - `foldl_add_eq_acc`: a foldl whose every step adds `0` reduces to
    the initial accumulator.
  - `foldl_indicator_unique`: a foldl summing an indicator function
    over a `List (Fin n)` with `Nodup` and one matching index reduces
    to `acc + 1`.
  - `foldl_indicator_square_eq`: squaring an indicator gives the same
    indicator (used by the diagonal case of `dotProduct_basis_basis`).
  - `foldl_dotProduct_basis_body`: strips `Vector.ofFn` lookups inside
    a dotProduct foldl body so subsequent rewrites can target the
    `if … then 1 else 0` form directly.
- Added `Matrix.dotProduct_basis_basis : dotProduct e_i e_j =
  if i = j then 1 else 0`. Proof: case split on `i = j`; the diagonal
  case uses `foldl_indicator_square_eq` followed by
  `foldl_indicator_unique`; the off-diagonal case shows every step's
  indicator product is `0` and applies `foldl_add_eq_acc`.
- Added `Matrix.gramMatrix_one : gramMatrix (1 : Matrix R n n) = 1` by
  rewriting each row of the identity as a basis-vector `Vector.ofFn`,
  unfolding `gramMatrix`, and applying `dotProduct_basis_basis`.
- Added `Matrix.submatrix_one : submatrix (1 : Matrix R n n) k = 1`
  via entry-wise extensionality and case-split on the entry equality.
- Added `Matrix.identity_independent : (1 : Matrix Int n n).independent`
  in `HexLLL/Basic.lean`. Three-line proof: rewrite via
  `gramMatrix_one`, `submatrix_one`, `det_one`; close with `decide`
  on `0 < 1`.
- Imported `Batteries.Data.List.Lemmas` in `HexMatrix/Basic.lean` to
  pull `List.nodup_finRange` and `List.mem_finRange` into scope.
