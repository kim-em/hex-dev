- The per-bad-vector resultant valuation `p^(a·d) ∣ Res(f, aux)` that

  is HO-4 **D1** (fast-path *succeeds* = performance), "not relied on by the rest of
  the project." Correctness rides Lemma 3.4 (exact-division certificate).

  `#coreFactors = #irreducibles(core)` fed to
  `UFDPartition.irreducible_of_partition_card_eq_normalizedFactors_card`
  (PartitionRefinement.lean:349). `count_le` is already proven from executable

  **forward** `W ⊆ L'` (`trueFactorIndicatorLattice_le_projectedRowSpan`, Lattice.lean:412,

  coarser*, so each emitted class ⊆ one true support, the emitted partition refines
