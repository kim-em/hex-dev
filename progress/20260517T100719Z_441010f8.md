### Scope investigation for #4617

Full proof is ~120-150 lines via `induction_on_coprime` over `p`,
producing 4 cases:

* **h0** (`p = 0`): trivial, ~5 lines. `derivative 0 = 0`,
  `gcd_zero_right`, `radical_zero`, `div_one`. Compiles locally.
* **h1** (`IsUnit p`): trivial, ~10 lines.
  `Polynomial.isUnit_iff.mp hu` gives `‚àÉ c, IsUnit c ‚àß C c = u`
  (destructure with `obtain ‚ü®c, _, rfl‚ü©`), then `derivative (C c) = 0`,
  `gcd u 0 = u`, `radical_of_isUnit` plus `div_one`. Compiles locally.
* **hpr.zero** (`q^0 = 1`): trivial, ~5 lines via `radical_one`. Compiles
  locally.
* **hpr.succ** (`q^(k+1)`, q prime): char-0 specific, ~50 lines.
  * Forward (`q^k ‚à£ gcd`): via
    `pow_sub_one_dvd_derivative_of_pow_dvd` + `EuclideanDomain.dvd_gcd`.
  * Reverse (`gcd ‚à£ q^k`): `dvd_prime_pow` gives `‚àÉ m ‚â§ k+1, Associated
    g (q^m)`; ruling out `m = k+1` uses `IsUnit.dvd_mul_left` to extract
    `q ‚à£ derivative q` after factoring `q^(k+1) ‚à£ C(k+1) * q^k * q'`,
    contradicted by `IsCoprime q (derivative q)` from
    `PerfectField.separable_of_irreducible` (`PerfectField.ofCharZero`
    is an instance) plus `IsCoprime.isUnit_of_dvd' dvd_rfl`.
  * `divRadical (q^(k+1)) ~ q^k`: via `radical_pow_of_prime` +
    `radical_mul_divRadical` + `Associated.of_mul_left` cancellation
    by `normalize q ‚âÝ 0` (from `normalize_eq_zero.not.mpr hq_ne`).
  * `C ((k:K)+1)` is a unit via `Polynomial.isUnit_C.mpr
    (isUnit_iff_ne_zero.mpr ...)` with `k+1 ‚âÝ 0` in char-0.
  * Compiled locally.
* **hcp** (`IsRelPrime x y` + IH): algebraic bottleneck. ~40-60 lines.
  * `IsRelPrime.isCoprime` gives `IsCoprime` in a PID.
  * `EuclideanDomain.divRadical_mul` decomposes the divRadical side.
  * Forward (`gxx' * gyy' ‚à£ g`): straightforward via
    `EuclideanDomain.dvd_gcd` plus `mul_dvd_mul` and the IH facts.
  * Reverse (`g ‚à£ gxx' * gyy'`): defining `gx := gcd g x`,
    `gy := gcd g y`, we can show `gx ‚à£ gcd x x'` (and symmetrically)
    using `IsCoprime.dvd_of_dvd_mul_right` (with `IsCoprime gx y`
    derived from `IsCoprime x y` plus `gx ‚à£ x`). The missing step is
    `g ‚à£ gx * gy`, which expresses the UFD fact "if `g ‚à£ x*y` and
    `IsCoprime x y` then `Associated g (gcd g x * gcd g y)`". I could
    not find a direct Mathlib lemma; multiplicity argument would work
    but is heavy.

Recommendation in skip comment: planner decompose into
