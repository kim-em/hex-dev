# Progress - feature #3846

## Accomplished

Wired up the unscaled derivative-active Yun residual provider in
`HexPolyFp/SquareFree.lean`:

- New helper
  `yunFactorsContributionResidualComplete_of_pairwise_reachable_levelCompletes`:
  takes a `yunFactorsPairwiseReachable c w fuel` chain plus a
  `yunFactorsLevelCompletes c w base level fuel` termination witness
  and returns `yunFactorsContributionResidualComplete c w multiplicity fuel`,
  closing each `isOne c = true` exit point with
  `yunFactorsPairwiseReachable_terminal_residual_derivative_zero`
  (landed via #3882 / #3864).
- New theorem
  `yunFactorsContributionResidualDerivativeZero_of_derivative_split`:
  discharges the unscaled contribution residual derivative-zero hypothesis
  on `(c, g) = (f / gcd f f', gcd f f')` from the size bound, the nonzero
  hypotheses on `f` and `f'`, and a derivative-active state provider.
  Builds `pairwise_reachable` and `derivativeActive_reachable` from
  `_of_derivative_split`, gets `LevelCompletes` via
  `yunFactorsLevelCompletes_of_size_bound_derivative_active` (avoids
  the `0 < multiplicity` requirement of the `_initial_split` wrapper),
  applies the helper, then converts via
  `yunFactorsContributionResidualDerivativeZero_of_complete`.
- New theorem `yunFactorsResidualDerivativeZero_of_derivative_split`:
  composes the contribution-level provider with
  `yunFactorsResidualDerivativeZero_of_derivative_split_contribution`.

The signatures take `hstate` for the derivative-active state predicate
(shared with the existing `_initial_split` wrapper). Downstream
`squareFreeAuxRev` callers already supply this, so the dependency is
free at the call site. No `axiom`, `native_decide`, `TODO`, `FIXME`,
or new theorem-level `sorry`; executable definitions unchanged.
