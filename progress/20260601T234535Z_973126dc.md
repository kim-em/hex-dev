# HexMatrix: one-row nMatrix setRow transports to ordered nDet (issue #6143)

## Accomplished

- Added two private row-equality helpers in `HexMatrix/Determinant.lean`:
  - `rowMoveUp_setRow_nMatrix_replace_first_eq_nMatrix`: for ordered
    `a < b < t`, replacing the `(t.val - 2)`-th row of `nMatrix B a b hab`
    by `B[a]` and sliding it up by `t.val - a.val - 2` adjacent swaps
    reproduces `nMatrix B b t hbt`.
  - `rowMoveUp_setRow_nMatrix_replace_second_eq_nMatrix`: same setup but
    inserting `B[b]` and sliding to `b.val - 1` by `t.val - b.val - 1`
    swaps, reproducing `nMatrix B a t (Nat.lt_trans hab hbt)`.
- Added the four issue-requested ordered four-row transports
  `det_setRow_nMatrix_r{2,3}_r{0,1}_eq_pow_mul_nDet_*`, each calling
  the matching helper, threading through `det_rowMoveUp`, and cancelling
  `(-1)^k * (-1)^k = 1` via the existing private `neg_one_pow_mul_self`.
- All names kept private; signatures match the issue body exactly.
